Formatando e Dando Semântica ao Texto

Para enfatizar algo, usamos texto em itálico (para voz ou terminologia) ou **texto com forte importância** (para conteúdo crucial). Se um texto foi deletado ou alterado, a tag `del` é útil. E para uma marcação de destaque relevante, a tag `mark` é perfeita.

Informações de copyright, avisos legais ou notas de rodapé geralmente usam a tag Texto pequeno - Direitos Reservados © 2025.

Números com potência como 22 ou fórmulas químicas como H2O usam as tags `` e `` respectivamente.

Podemos sublinhar textos com a tag sublinhado (embora o uso de CSS `text-decoration: underline;` seja preferível para fins de estilo).

Para abreviações, use a tag HTML para fornecer o significado completo ao passar o mouse.

Tags Depreciadas e Acessibilidade

Alguns elementos HTML, como a tag `font`, foram depreciadas em HTML5 e não devem ser usadas para estilização direta. Para controlar a aparência do texto, o **CSS** é a ferramenta correta e recomendada. Ignorar isso prejudica a manutenção e a compatibilidade futura.
